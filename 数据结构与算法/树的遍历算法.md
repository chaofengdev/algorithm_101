# 深度优先遍历DFS

## 先序遍历

方法1：非递归--中序遍历统一版本

```java
class Solution {
    public List<Integer> preorderTraversal(TreeNode root) {
        //结果集合
        List<Integer> res = new LinkedList<>();
        //辅助栈
        Stack<TreeNode> stack = new Stack<>();
        //判空
        if(root == null) return res;
        //工作指针
        TreeNode cur = root;
        //循环遍历
        while(!stack.isEmpty() || cur != null) {
            while(cur != null) {
                res.add(cur.val);//访问根节点--访问的位置要清楚
                stack.push(cur);//根节点入栈
                cur = cur.left;//工作指针指向该结点的左子树
            }
            TreeNode node = stack.pop();//栈中弹出结点元素（当然这里可以使用cur来引用）
            cur = node.right;//工作指针指向该结点的右子树
        }
        //返回结果集
        return res;
    }
}
```

方法2：非递归--先序独立版本

```java
class Solution {
    public List<Integer> preorderTraversal(TreeNode root) {
        //结果集合
        List<Integer> res = new LinkedList<>();
        //辅助栈
        Stack<TreeNode> stack = new Stack<>();
        //判空
        if(root == null) return res;
        //根节点入栈
        stack.push(root);
        //循环
        while(!stack.isEmpty()) {
            TreeNode node = stack.pop();
            res.add(node.val);//访问结点
            if(node.right != null) stack.push(node.right);//右子结点入栈
            if(node.left != null) stack.push(node.left);//左子结点入栈
        }
        //返回结果集
        return res;
    }
}
```

方法3：递归

```java
class Solution {
    List<Integer> res = new ArrayList<>();
    public List<Integer> preorderTraversal(TreeNode root) {
        dfs(root);
        return res;
    }
    //先序遍历
    public void dfs(TreeNode root) {
        if(root == null) return;
        res.add(root.val);
        dfs(root.left);
        dfs(root.right);
    }
}
```
