# 广度优先遍历BFS

方法1：非递归

```java
class Solution {
    public List<List<Integer>> levelOrder(TreeNode root) {
        //定义结果集合
        List<List<Integer>> res = new ArrayList<>();
        //定义辅助队列
        Deque<TreeNode> queue = new ArrayDeque<>();
        //判空
        if(root == null){
            return res;
        }
        //根节点入队列
        queue.offer(root);
        //核心逻辑
        while(!queue.isEmpty()){
            //队列长度
            int queue_size = queue.size();
            //定义辅助集合
            List<Integer> list = new ArrayList<>();
            //“遍历queue”中所有元素，可以认为它们都是在一层的。
            for(int i = 0; i < queue_size; i++){
                //取出此刻队头元素
                TreeNode node = queue.poll();
                //将左右子节点加入队列
                if(node.left != null){
                    queue.offer(node.left);
                }
                if(node.right != null){
                    queue.offer(node.right);
                }
                //将队头元素加入辅助集合list
                list.add(node.val);
            }
            res.add(list);
        }
        //返回结果集
        return res;
    }
}
```

# 深度优先遍历DFS

## 先序遍历

方法1：非递归--中序遍历统一版本

```java
class Solution {
    public List<Integer> preorderTraversal(TreeNode root) {
        //结果集合
        List<Integer> res = new LinkedList<>();
        //辅助栈
        Stack<TreeNode> stack = new Stack<>();
        //判空
        if(root == null) return res;
        //工作指针
        TreeNode cur = root;
        //循环遍历
        while(!stack.isEmpty() || cur != null) {
            while(cur != null) {
                res.add(cur.val);//访问根节点--访问的位置要清楚
                stack.push(cur);//根节点入栈
                cur = cur.left;//工作指针指向该结点的左子树
            }
            TreeNode node = stack.pop();//栈中弹出结点元素（当然这里可以使用cur来引用）
            cur = node.right;//工作指针指向该结点的右子树
        }
        //返回结果集
        return res;
    }
}
```

方法2：非递归--先序独立版本

```java
class Solution {
    public List<Integer> preorderTraversal(TreeNode root) {
        //结果集合
        List<Integer> res = new LinkedList<>();
        //辅助栈
        Stack<TreeNode> stack = new Stack<>();
        //判空
        if(root == null) return res;
        //根节点入栈
        stack.push(root);
        //循环
        while(!stack.isEmpty()) {
            TreeNode node = stack.pop();
            res.add(node.val);//访问结点
            if(node.right != null) stack.push(node.right);//右子结点入栈
            if(node.left != null) stack.push(node.left);//左子结点入栈
        }
        //返回结果集
        return res;
    }
}
```

方法3：递归

```java
class Solution {
    List<Integer> res = new ArrayList<>();

        dfs(root);
        return res;
    }
    //先序遍历
    public void dfs(TreeNode root) {
        if(root == null) return;
        res.add(root.val);
        dfs(root.left);
        dfs(root.right);
    }
```

## 后序遍历

方法1：非递归--先序遍历统一版本

> 这个方法比较巧妙，使用Collections.reverse(res)，将原本[根右左]翻转成[左右根]。
>
> 这个方法的问题在于，本质上还是异构的先序遍历变种，通过两次反转，将结果反转称为正确的序列，但本质上不是后序遍历。



```java
class Solution {
    public List<Integer> postorderTraversal(TreeNode root) {
        List<Integer> res = new ArrayList<>();
        Stack<TreeNode> stack = new Stack<>();
        stack.push(root);
        while(!stack.isEmpty()) {
            TreeNode node = stack.pop();
            res.add(node.val);
            if(node.left != null) stack.push(node.left);
            if(node.right != null) stack.push(node.right);
        }
        Collections.reverse(res);
        return res;
    }
}
```

方法2：非递归--中序遍历统一版本

> 注意pre的定义：用来标识前一个已经被访问的结点。

```java
class Solution {
    public List<Integer> postorderTraversal(TreeNode root) {
        //结果集合与辅助栈
        List<Integer> res = new LinkedList<>();
        Deque<TreeNode> stack = new LinkedList<>();
        //定义指针，用来标识前一个访问结点，初始为空
        TreeNode pre = null;
        //工作指针，用来标识当前访问结点，初试为root
        TreeNode cur = root;
        //核心逻辑
        while(cur != null || !stack.isEmpty()){//注意cur不为空的原因：遍历到根节点的右子节点时stack为空
            //子节点一路压栈
            while(cur != null){
                stack.push(cur);
                cur = cur.left;
            }
            //出栈
            TreeNode node = stack.pop();
            //本题关键：当前结点的右指针为空或者右指针已经访问完毕，则将结点加入结果集合res，同时更新pre与cur
            if(node.right == null || pre == node.right){
                res.add(node.val);
                pre = node;
                cur = null;
            }else{
                //如果当前结点有右子节点，则重新将其压栈，并将指针指向右子节点
                //原因：后序遍历要求先左右子节点后根节点，所以根节点需要重新压栈
                stack.push(node);
                cur = node.right;
            }
        }
        //返回结果集合
        return res;
    }
}
```
