## 斐波那契数列

方法1：动态规划

```java
import java.util.Scanner;

// 注意类名必须为 Main, 不要有任何 package xxx 信息
public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int num = sc.nextInt();
        int[] dp = new int[num + 1];
        dp[0] = 0;
        dp[1] = 1;
        for(int i = 2; i < num + 1; i++) {
            dp[i] = dp[i - 2] + dp[i - 1];
        }
        System.out.println(dp[num]);
    }
}
```

方法2：优化空间复杂度

```java
import java.util.Scanner;

// 注意类名必须为 Main, 不要有任何 package xxx 信息
public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int num = sc.nextInt();
        int[] dp = new int[num + 1];
        // dp[0] = 0;
        // dp[1] = 1;
        int pre = 0;
        int cur = 1;
        for(int i = 2; i < num + 1; i++) {
            // dp[i] = dp[i - 2] + dp[i - 1];
            int temp = pre + cur;
            pre = cur;
            cur = temp;
        }
        // System.out.println(dp[num]);
        System.out.println(cur);
    }
}
```

## 跳台阶

方法1：动态规划

```java
import java.util.Scanner;

// 注意类名必须为 Main, 不要有任何 package xxx 信息
public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int num = sc.nextInt();
        int[] dp = new int[num + 1];
        dp[0] = 0;
        dp[1] = 1;
        if(num > 1) dp[2] = 2;
        for(int i = 3; i < num + 1; i++) {
            dp[i] = dp[i - 1] + dp[i - 2];
        }
        System.out.println(dp[num]);
    }
}
```
